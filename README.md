# 🐍 Fundamentos de Python
Este repositório foi criado para documentar toda a minha jornada de aprendizado em Python. Aqui, concentro meus estudos teóricos, anotações conceituais e a resolução de exercícios práticos.

⭐ Este repositório é um projeto em construção. À medida que avanço nos módulos de estudo, novos exercícios, desafios e anotações serão adicionados regularmente para registrar minha evolução técnica.

Para os meus estudos, utilizo o Jupyter Notebook integrado ao PyCharm, o que me permite unir a documentação em texto (Markdown) com a execução de código em tempo real.
